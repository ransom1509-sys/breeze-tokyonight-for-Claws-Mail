# Breeze Tokyo Night for Claws-Mail
## About
This is the "Breeze Tokyo Night" theme for Claws Mail.
This theme is a modification of the original "Breeze" 
theme for Claws-Mail designed by Andreas Kainz

Changes to the original theme:\
"Tokyo Night" colors\
IMAP icons added.\
Zoom icons added.\
Some misspelled icon names fixed.

## Installation
Download the [compressed archive](theme/breeze-tokyonight.tar.gz).\
Extract the archive in your themes folder (~/.claws-mail/themes or /usr/share/claws-mail/themes).\
In Claws-Mail select Configuration -> Settings -> Themes and choose "breeze-tokyonight.
